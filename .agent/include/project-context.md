# Project Context

## What This Repository Is

- Repository: `agam-lang/governance`
- Purpose: Define and maintain the organizational policies, team structure, decision-making processes, and community standards for the agam-lang organization.
- Technology: Markdown policy documents

## Organization Context

- This is part of the `agam-lang` ecosystem.
- The core Agam compiler lives at `agam-lang/agam`.
- The standard library lives at `agam-lang/std`.
- The scientific platform lives at `agam-lang/agamlab`.
